# Intercept

## Install
1. Clone repository
   ```sh
   git clone https://github.com/EmperorWasTaken/Intercept.git
   cd Intercept
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Build the extension
   ```sh
   npm run build
   ```
4. Load in Edge
   - Open `edge://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the `/dist` folder from the repository
